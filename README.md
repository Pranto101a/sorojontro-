# ষড়যন্ত্র (Shorojontro) — Render ডিপ্লয় + অনলাইন মাল্টিপ্লেয়ার

এই প্রজেক্টে:
- `index.html` (ওয়েব গেম)
- `images/` (অ্যাসেট)
- `server.js` (Express + PeerJS signaling server)
- `manifest.webmanifest` + `sw.js` (Android/PWA ইনস্টল-ফ্রেন্ডলি)

## ১) লোকালি রান (টেস্ট)

```bash
npm install
npm start
```

তারপর ব্রাউজারে খুলুন: `http://localhost:3000`

## ২) Render-এ ডিপ্লয় (Recommended)

Render-এ একটি **Web Service** তৈরি করুন।

**Build Command**
```bash
npm install
```

**Start Command**
```bash
npm start
```

ডিপ্লয় হওয়ার পর:
- গেম URL ওপেন করে **🌐 অনলাইন ম্যাচ হোস্ট** দিন
- বন্ধুদেরকে **রুম কোড** দিন
- তারা একই URL থেকে **🔗 কোড দিয়ে যোগ দিন** দিয়ে জয়েন করবে

## ৩) “সব প্লেয়ারের নাম দেখা যাচ্ছে না” সমস্যার ফিক্স

লোকাল/AI মোডে আগে শুধু একটি নাম ইনপুট ছিল। এখন **সব খেলোয়াড়ের নাম** সেকশনে (লবি স্ক্রিনে) ২–৬ জনের নাম সেট করা যাবে।

## ৪) UI + Sound

- নতুন ব্যাকগ্রাউন্ড (`assets/bg.jpg`) যোগ করা হয়েছে
- Home/Game স্ক্রিনের জন্য আলাদা ব্যাকগ্রাউন্ড (`assets/home_bg.jpg`, `assets/game_bg.jpg`) যোগ করা হয়েছে
- টপবারে **🔊 সাউন্ড** বাটন আছে (অন/অফ) — ক্লিক/মডাল/টার্ন সাউন্ড এফেক্ট যোগ করা হয়েছে

> নোট: যদি আপনার কাছে আসল UI অ্যাসেট (PNG/JPG) থাকে, এই `assets/*_bg.jpg` ফাইলগুলো রিপ্লেস করলেই UI আরো একদম আপনার রেফারেন্সের মতো হবে।
